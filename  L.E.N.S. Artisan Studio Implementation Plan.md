  Implementation Flow: Artisan Studio Journey                                                                                                                                                                  
                                                                                                                                                                                                               
  The core feature being built is the guided listing flow for blind/low-vision artisans. Here's the step-by-step implementation:                                                                               
                                                                                                                                                                                                               
  Phase 1: Entry & Setup                                                                                                                                                                                       
                                                                                                                                                                                                               
  HomePage → "Enter Artisan Studio" button → SellMode component                                                                                                                                                
                                                                                                                                                                                                               
  Key user actions:                                                                                                                                                                                            
  1. User clicks "Enter Artisan Studio" from homepage                                                                                                                                                          
  2. App announces: "Welcome to L.E.N.S. Artisan Studio. Voice coaching ready."                                                                                                                                
  3. Shows "Start Guided Listing" button or voice command prompt                                                                                                                                               
                                                                                                                                                                                                               
  ---                                                                                                                                                                                                          
  Phase 2: First Photo Capture                                                                                                                                                                                 
                                                                                                                                                                                                               
  Voice command "Take photo" OR tap button → Camera opens                                                                                                                                                      
                                                                                                                                                                                                               
  Flow:                                                                                                                                                                                                        
  1. TTS announces: "Opening camera. Position your craft in good light. Tap to capture."                                                                                                                       
  2. User positions phone over craft (e.g., hand-knit sweater)                                                                                                                                                 
  3. User taps to capture photo                                                                                                                                                                                
  4. TTS announces: "Got your photo. Analyzing with Gemma now. This takes a moment..."                                                                                                                         
                                                                                                                                                                                                               
  ---                                                                                                                                                                                                          
  Phase 3: First Analysis                                                                                                                                                                                      
                                                                                                                                                                                                               
  Upload to Gemma 4 → Get critique → Speak results                                                                                                                                                             
                                                                                                                                                                                                               
  Analysis response structure:                                                                                                                                                                                 
  - What's working: "Your hand-knit sweater shows beautiful cable texture. The cream color reads warmly."                                                                                                      
  - What needs fixing: "However, the lighting is uneven — the left side is darker than the right. There's a shadow across the collar."                                                                         
  - Specific action: "Move closer to the window and rotate the sweater so light falls evenly across it."                                                                                                       
                                                                                                                                                                                                               
  Implementation notes:                                                                                                                                                                                        
  - For demo: Edit 45-50s local inference wait down to 5s                                                                                                                                                      
  - Show "Analyzing with Gemma..." animation during wait                                                                                                                                                       
  - Use real Gemma 4 responses for authenticity                                                                                                                                                                
                                                                                                                                                                                                               
  ---                                                                                                                                                                                                          
  Phase 4: Retry Loop                                                                                                                                                                                          
                                                                                                                                                                                                               
  Analysis results → Retake prompt → Second photo → Compare                                                                                                                                                    
                                                                                                                                                                                                               
  Flow:                                                                                                                                                                                                        
  1. TTS asks: "Would you like to try again with this tip? Say 'yes' or tap Retake."                                                                                                                           
  2. User says "Yes" or taps Retake                                                                                                                                                                            
  3. TTS reminds: "Opening camera again. Remember: move closer to the window and rotate so light falls evenly."                                                                                                
  4. User captures second photo                                                                                                                                                                                
  5. TTS announces: "Got your second photo. Now comparing both shots to find the winner..."                                                                                                                    
                                                                                                                                                                                                               
  ---                                                                                                                                                                                                          
  Phase 5: Comparison & Winner                                                                                                                                                                                 
                                                                                                                                                                                                               
  Compare both photos → Pick winner → Announce                                                                                                                                                                 
                                                                                                                                                                                                               
  Comparison UI:                                                                                                                                                                                               
  - Side-by-side flash of both photos (2 seconds)                                                                                                                                                              
  - "Comparing..." animation (edited to ~5s)                                                                                                                                                                   
  - Winner badge appears on Photo 2                                                                                                                                                                            
  - Subtle confetti/highlight animation                                                                                                                                                                        
                                                                                                                                                                                                               
  TTS winner announcement:                                                                                                                                                                                     
  "Comparison complete! Photo two is the winner!                                                                                                                                                               
   The lighting is now even across the entire sweater.                                                                                                                                                         
   The cable-knit texture is crisp and clear.                                                                                                                                                                  
   The cream color looks natural and warm.                                                                                                                                                                     
   This photo is marketplace-ready."                                                                                                                                                                           
                                                                                                                                                                                                               
  ---                                                                                                                                                                                                          
  Phase 6: Listing Generation                                                                                                                                                                                  
                                                                                                                                                                                                               
  Winner selected → Generate Etsy listing → Copy to clipboard                                                                                                                                                  
                                                                                                                                                                                                               
  Flow:                                                                                                                                                                                                        
  1. TTS asks: "Would you like me to generate your listing?"                                                                                                                                                   
  2. User says "Yes" or taps button                                                                                                                                                                            
  3. TTS announces: "Generating your Etsy listing..."                                                                                                                                                          
  4. Listing appears with:                                                                                                                                                                                     
    - Title: "Hand-knit cable sweater in natural cream"                                                                                                                                                        
    - Description: "Cozy hand-knit sweater featuring classic cable-knit pattern. Made with soft natural wool. Perfect for cool autumn mornings. Handcrafted with care."                                        
    - Hashtags: #handmade #knitting #wool #cable-knit #sweater #artisan #cozy #handcrafted                                                                                                                     
  5. TTS reads listing: "Here's your listing: [reads title, description, hashtags]. Say 'copy' or tap to copy everything."                                                                                     
                                                                                                                                                                                                               
  ---                                                                                                                                                                                                          
  Phase 7: Copy & Export                                                                                                                                                                                       
                                                                                                                                                                                                               
  Listing ready → Copy to clipboard → Open Etsy (out of scope)                                                                                                                                                 
                                                                                                                                                                                                               
  Flow:                                                                                                                                                                                                        
  1. User says "Copy" or taps "Copy All for Etsy" button                                                                                                                                                       
  2. Button animates (pressed state)                                                                                                                                                                           
  3. Checkmark/success confirmation shows                                                                                                                                                                      
  4. TTS confirms: "Copied! Your photo and listing are ready. Open Etsy and paste to create your listing."
